iKey Product Facts at a Glance

DeveloperAgile Offroad (Ineos-specific). Internal software — not a re-badged 3rd-party engine.
Price (target)~$499 USD incl. USB dongle/OBDII reader + 2 yrs free updates. Renewal after 2 yrs explicitly not another $499 (TBD, minimal). Family plan ~+$100/extra Grenadier. Pro/dealer tier planned.
Anticipated launch17 July 2026 (still in beta — Beta 3.5 as of early June).
PlatformWindows 10/11 only at launch, registered to one PC by UUID. Mac only via VM/UTM workarounds. Multi-platform = future aspiration.
HardwareWired USB-to-OBD dongle (Tactrix OpenPort 2.0 / clone type) supplied in bundle; ~200 units in initial stock.
ConnectivityInternet for license auth, OTA updates, security-key calc, remote support & report send. DTC read/reset + basic functions work offline.
LicensingVIN-locked, transferable to a new VIN. Software keeps running after 2-yr window (only updates/support lapse).
Engine coveragePetrol (BMW B58) working in beta; Diesel (B57) testers just onboarded (AU & UK), branching by Euro emissions market.
Named competitorGlobeGuard (Switzerland) OBD2 service tool ≈ $1,036; Agile aims to undercut it at ~half price with more function. Users also benchmark Ross-Tech VCDS, BimmerCode, Autel.

⊘6 · Missing / Weaker vs. Third-Party Tools for Competitive Vehicles GAP ANALYSIS

The Grenadier shares BMW B58 petrol / B57 diesel engines and the ZF8 transmission, so the natural competitive set is BMW/VAG-oriented 3rd-party tools — Autel MaxiSys, Ross-Tech VCDS, OBDeleven, Carly, ABRITES, GAP Diagnostic, BimmerCode, plus tuning tools (bootmod3, MHD, xHP). Items below are features those tools commonly offer that the Agile tool currently lacks, only has on its roadmap, or does in a more limited way.
